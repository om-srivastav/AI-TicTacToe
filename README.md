# 🤖 AI Tic Tac Toe

An intelligent Tic Tac Toe game powered by **Google Gemini AI**, built with **Python** and **Streamlit**.

Challenge the AI, play with a friend, and experience a modern, responsive interface.

---

## ✨ Features

- 🤖 Human vs Gemini AI
- 👥 Human vs Human Mode
- 🎯 Smart AI Move Generation using Gemini
- 📊 Live Scoreboard
- 🔄 One-click Game Reset
- 🎨 Modern Responsive UI
- ⚡ Fast and Lightweight
- 🌐 Ready for Streamlit Cloud Deployment

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini 2.5 Flash API

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/om-srivastav/AI-TicTacToe.git
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Add your Gemini API Key

Create a `.env` file:

```env
GOOGLE_API_KEY=YOUR_API_KEY
```

### Run the application

```bash
streamlit run app_v3.py
```
